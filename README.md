# 🏃‍♂️ Carrera Atlética - Programación Concurrente

Proyecto desarrollado en Java que simula una carrera entre 5 corredores utilizando hilos (Threads).

## 🚀 Funcionalidades
- Registro de corredores
- Simulación de carrera con hilos
- Resultados en orden de llegada
- Interfaz gráfica con Java Swing

## 🛠️ Tecnologías
- Java
- NetBeans
- Swing

## 🎯 Objetivo
Aplicar el uso de programación concurrente mediante hilos para simular procesos en ejecución simultánea.
